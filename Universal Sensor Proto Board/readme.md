# Universal Sensor Proto Board
An universal sensor prototyping breakout board with pull-up or pull-down resistors  
that can be used to interface a variety of sensors like - 
* LDR
* LM35
* DHT11
* DHT12
* NTC
* IR Sensor
* Flex sensor
* Hall Effect sensor
